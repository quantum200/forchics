# Сторінка продукту ForChics
## 1. Огляд та Основні Технології

Цей проєкт реалізує висококонверсійний дизайн сторінки продукту з інтерактивними елементами та динамічним відгуком на прокручування.

| Технологія | Призначення |
| :--- | :--- |
| **HTML5** | Семантична структура та модульна розбивка контенту. |
| **CSS3** | Flexbox, CSS Grid, `position: sticky` для складних макетів. |
| **JavaScript (ES6+)** | Intersection Observer, `localStorage`, `setTimeout` для анімації та логіки. |
| **Бібліотеки** | jQuery, Slick Slider (для каруселі відгуків). |

## 2. Структура Файлів та Модульність CSS

CSS-код розділено на модулі (`header_css.css`, `main_css.css` тощо), і вони підключаються в `<head>`. JavaScript-логіка розділена на окремі модулі та підключається в `<body>` після завантаження DOM.

### Критичні елементи HTML:

* **`meta name="viewport"`**: **Обов'язково** присутній у `<head>` для коректного мобільного масштабування.
  ```html
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  ```
* **Footer JS Order**: Порядок підключення JS у `<body>` суворо дотриманий (jQuery → Slick → інші скрипти).

## 3. Реалізований Функціонал

### 3.1. Адаптивність (Responsive Design)

* **Стратегія:** Mobile-First з переходом на десктоп.
* **Контрольні точки:** $1000\text{px}$ (Малий Десктоп/Планшет), $768\text{px}$ (Планшет/Мобільний), $320\text{px}$ (Малий Мобільний).
* **Основний макет:** Перехід з $2$ колонок (`left-side`, `right-side`) на **$1$ вертикальний стовпець** на екранах менших за $1000\text{px}$.
* **Футер:** Адаптований макет колонок з $5$ елементів до схеми **3-в-ряд** ($1000\text{px}$) та **2-в-ряд** ($768\text{px}$).

### 3.2. Динамічні елементи (JavaScript)

| Функціонал | Механізм | Опис |
| :--- | :--- | :--- |
| **Таймер зворотного відліку** | `setInterval`, `localStorage` | Відлік від $24$ годин, зберігає час закінчення після оновлення сторінки. |
| **Sticky Bar (Нижня плашка)** | Intersection Observer | Фіксована плашка "Додати до кошика" з'являється, коли користувач прокручує повз блок `.main-text`, і зникає, коли видно `<footer>`. |
| **Анімація прокрутки** | Intersection Observer | Плавно "виїжджає" контент (іконки USP, тижні використання) по черзі, коли блок потрапляє в область видимості. |
| **Зміна зображень** | JS `mouseover`/`mouseout` | Іконки соціальних мереж та зображення товару змінюють `src` на `[name]_full.png` при наведенні. |
| **Галерея продукту** | JS Event Listeners | Кліки по мініатюрах (`.thumbnail`) змінюють головне зображення (`#mainImage`). |
| **Навігація в галереї** | JS Mousemove/Click | Кліки по лівій/правій половині головного зображення перемикають фотографії (наступне/попереднє). |
| **Аккордеон** | JS `classList.toggle('active')` | Незалежне розкриття блоків (FAQ) та побудова анімації тексту "построково" (`.animated-line`) для "Гарантії". |
| **Мобільні фільтри** | JS `is-active` toggle | Фільтри ховаються за кнопкою `More Filters` на екранах $< 768\text{px}$ і розкриваються по кліку. |
